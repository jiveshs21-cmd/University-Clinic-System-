# Project Plan


## Project Title
University Clinic Appointment System

## Project Objective
To develop a web-based system that allows university students to book clinic appointments online.

## Project Scope
The system will allow students to:
- Login to the system
- Book clinic appointments
- View clinic schedules

Clinic staff will be able to:
- Manage appointments
- Update doctor availability

## Timeline
Week 1: Planning
Week 2: System Design
Week 3: Prototype Development
Week 4: Testing
Week 5: Final Report
