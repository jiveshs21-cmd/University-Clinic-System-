# User Stories

## Student


As a student, I want to create an account so that I can access the university clinic system.

As a student, I want to log into the clinic system so that I can manage my healthcare appointments.

As a student, I want to view available doctors and clinic schedules so that I can choose a suitable appointment time.

As a student, I want to book a clinic appointment so that I can receive medical consultation.

As a student, I want to receive confirmation of my appointment so that I know my booking was successful.

As a student, I want to view my appointment history so that I can keep track of my medical visits.

As a student, I want to cancel or reschedule an appointment so that I can manage changes in my schedule.

As a student, I want to receive notifications or reminders before my appointment so that I do not miss it.

As a student, I want to access health information provided by the clinic so that I can improve my well-being.

---

## Clinic Staff


As a clinic staff member, I want to log into the system so that I can manage clinic operations.

As a clinic staff member, I want to view all appointment bookings so that I can organize the clinic schedule.

As a clinic staff member, I want to approve or manage appointment requests so that patient flow remains organized.

As a clinic staff member, I want to update doctor availability so that students can book accurate time slots.

As a clinic staff member, I want to access patient appointment records so that I can assist doctors during consultations.

---

## Administrator


As an administrator, I want to manage user accounts so that only authorized users access the system.

As an administrator, I want to monitor system usage so that the clinic platform runs smoothly.

As an administrator, I want to generate reports on appointments and system activity so that the university can evaluate clinic performance.
