# Final Report
University Clinic System


## Project Overview
The University Clinic System is a web-based platform designed to help university students access healthcare services efficiently. The system allows students to book clinic appointments, view doctor schedules, and manage their appointments online.


## Objectives
- Improve access to university healthcare services
- Reduce waiting time at the clinic
- Provide an organized appointment management system


## System Features
- User registration and login
- Book clinic appointments
- View and cancel appointments
- Doctors update appointment status
- Admin manages users and doctors


## Technologies Used
- Draw.io for system diagrams
- GitHub for version control
- Markdown for documentation


## Conclusion
The University Clinic System improves clinic management and provides students with an easier way to access healthcare services within the university environment.
