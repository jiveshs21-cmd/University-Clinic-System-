# Project Summary

## Project Title
University Clinic Appointment System


## Sustainable Development Goal
SDG 3: Good Health and Well-being


## Project Description
The University Clinic System is designed to improve healthcare services within the university. The system allows students to book clinic appointments online, check doctor schedules, and manage their appointments easily. It also allows doctors and administrators to manage schedules and users efficiently.


## Key Features
- User registration and login
- Book clinic appointments
- View and cancel appointments
- Doctors update appointment status
- Admin manages users and doctors
- View doctor schedules


## System Actors
- Student
- Doctor
- Administrator


## Prototype
The system prototype was designed to demonstrate the user interface and main functions of the University Clinic System.

The prototype includes the following screens:

- Login Page – allows users to securely access the system.
- Student Dashboard – displays available options for students.
- Book Appointment Page – allows students to select a doctor, date, and time for appointments.
- Doctor Schedule Page – shows doctor availability.
- Admin Management Page – allows administrators to manage users and doctors.

These prototype screens help visualize how users interact with the system before actual development.


## Project Phases
1. Planning – requirements gathering and project documentation.
2. Design – system diagrams and prototype design.
3. Testing – defining test cases and validating system functionality.
4. Final – project summary and documentation.


## Outcome
The project successfully demonstrates the planning, design, and testing of a university clinic management system using software engineering principles. The system provides a structured approach to managing clinic appointments and improving healthcare accessibility for students.
