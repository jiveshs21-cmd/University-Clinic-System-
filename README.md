# University-Clinic-System-

A web-based platform designed to help university students easily access healthcare services on campus. The system allows students to book medical appointments, view clinic schedules, access health information, and communicate with clinic staff. The website improves healthcare accessibility while reducing waiting time at the university clinic.

Project Title

University Clinic System

Sustainable Development Goal Chosen
SDG 3: Good Health and Well-being

This project supports the United Nations Sustainable Development Goal 3 by encouraging students to maintain better physical and mental health through easy access to healthcare services.


Project Description

The University Clinic Website is designed to digitize and simplify healthcare services for students. Currently, many university clinics require students to physically visit the clinic to book appointments, which often leads to long waiting times and inefficient scheduling.

This system allows students to log in using their university credentials and easily book appointments with available doctors. Students can check clinic schedules, view available services, and receive important health updates through the platform.

Clinic staff can manage appointments, update doctor schedules, maintain patient visit records, and monitor clinic activity through an administrative interface.

By implementing this system, the university clinic can improve service efficiency, reduce administrative workload, and provide faster healthcare access to students.

Key Features
Student Features

• Register and login using university credentials
• Book clinic appointments online
• View doctor availability and clinic schedule
• Access health information and announcements
• View appointment history


Clinic Staff Features

• Manage appointment bookings
• Update doctor schedules
• Maintain patient visit records
• Send clinic announcements


Administrator Features

• Manage user accounts
• Monitor system activity
• Generate clinic reports


System Actors

Student – books appointments and accesses health information

Clinic Staff / Doctor – manages appointments and patient records

Administrator – manages the system and users
